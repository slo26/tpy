Database Administration Tool for Drupal.

Based on Adminer (https://www.adminer.org).
Adminer (formerly phpMinAdmin) is a full-featured database management tool written in PHP.
Conversely to phpMyAdmin, it consist of a single file ready to deploy to the target server.

Works with MySQL, MariaDB, PostgreSQL, SQLite, MS SQL, Oracle, SimpleDB, Elasticsearch, MongoDB.
Supports PHP 7 with enabled sessions.
Available in many languages including Ukrainian (41).
Free for commercial and non-commercial use (Apache License or GPL 2).
